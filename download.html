v1.0-beta.1 [f3f22c3] - First Release 01/01/2016

Introduction

This is the first release of my Node.js port of BitCannon.

What works

  • Importing from RSS feeds

What doesn't Work

  • Probably lots of things. This is the first release so I expect there to be a lot of bugs.

Important Notes:

  • The binary needs to be run from the command line (Terminal.app on OS X, CMD on Windows and Whatever on GNU/Linux).
  • The config file is located in the resources/app folder
  • If you use command line parameters (they aren't yet documented but I intend to document them) the first command line parameter is ignored for some reason (I think this might be a bug with Electron)
